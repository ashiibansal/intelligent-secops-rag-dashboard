# Test Files for CSE-CIC-IDS2018 Models

## Files:
- **test_data.csv**: Complete test set with mixed benign/attack samples
- **benign_test.csv**: Only benign traffic samples (100 samples)
- **attack_test.csv**: Only attack traffic samples (100 samples)
- **quick_test.csv**: Small sample for quick testing (50 samples)

## Usage:
Upload any CSV file to the Streamlit application to test anomaly detection.

## Format:
All files contain network flow features with a 'Label' column showing ground truth.
